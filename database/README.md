# База данных — Корочки.есть

Используется **SQLite** (файл `data/korochki.db`). Отдельный сервер MySQL не требуется.

**ER-диаграмма для задания:** см. [ER-diagram.md](ER-diagram.md) и [ER-dbdiagram.dbml](ER-dbdiagram.dbml).

## Таблицы

| Таблица | Назначение |
|---------|------------|
| `users` | Пользователи |
| `applications` | Заявки на курсы |
| `feedbacks` | Отзывы (один на пользователя) |
| `admins` | Администраторы |

## Инициализация

При `npm start` схема и учётная запись администратора создаются автоматически.

Принудительная инициализация:

```bash
npm run db:init
```

Администратор: **Admin** / **KorokNET**

## Сброс данных

Удалите файл `data/korochki.db` и снова выполните `npm start` или `npm run db:init`.
